# BMI Calculator

## Description
This is a simple Body Mass Index (BMI) calculator that helps users determine their BMI based on their height and weight inputs. It also provides a basic health assessment based on the calculated BMI.

## Features
- Input fields for height (in cm) and weight (in kg).
- A "Calculate BMI" button to compute the result.
- Displays the calculated BMI along with a health assessment message (e.g., underweight, normal, overweight).
- Minimalistic dark-themed design with red accents.

## Technologies Used
- **HTML**: For structuring the BMI calculator layout.
- **CSS**: For styling the form and results display.
- **JavaScript**: For implementing the BMI calculation logic.

## How to Use
1. Clone this repository or download the code.
2. Open the `index.html` file in your browser.
3. Enter your height and weight in the respective input fields and click "Calculate BMI."
